# md_tasklist — GFM task-list demo

A short description of what is to be done, followed by a check-list.

## Morning routine

- [x] make coffee
- [x] feed the cat
- [ ] walk the dog
- [ ] read the newspaper

## Release checklist

- [ ] run the test suite
- [ ] bump the version in `pyproject.toml`
- [ ] update `CHANGELOG.md`
- [x] open the release PR
- [ ] merge once CI is green

## Mixed with normal bullets

Regular bullets and task items can coexist in a single list — this is a
worked example:

- alpha (plain bullet)
- [ ] beta (task, open)
- [x] gamma (task, done)
- delta (plain bullet)
