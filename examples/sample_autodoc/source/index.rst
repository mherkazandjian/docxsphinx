sample_autodoc — Sphinx autodoc rendered to Word
=================================================

This example invokes :mod:`sphinx.ext.autodoc` against the in-repo
:mod:`mymodule` to verify that docxsphinx emits Word output for the full
set of ``desc_*`` description nodes Sphinx produces for Python objects:
class signatures, method signatures, parameter lists, type annotations,
return types, and docstring field lists.
